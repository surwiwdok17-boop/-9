# 9 Лабораторна робота
Вебзастосунок Магазин

Опис проєкту та його призначення

Вебзастосунок реалізує функціонал онлайн‑магазину з можливістю перегляду товарів, додавання їх у кошик, оформлення замовлень та залишення відгуків. Проєкт створений для навчальних цілей, демонструє роботу з Flask, SQLAlchemy, шаблонами Jinja2 та інтеграцією Swagger для API документації.

Використані технології та обґрунтування вибору

Flask — легкий веб‑фреймворк для швидкої розробки.

SQLAlchemy ORM — зручна робота з базою даних.

SQLite — проста база даних для локальної розробки.

Tailwind CSS — сучасна бібліотека стилів для адаптивного UI.

Flasgger (Swagger UI) — автоматична документація API.

dotenv — для конфігурації через змінні оточення.

Архітектура системи

Frontend: HTML + Jinja2 шаблони, стилізація Tailwind.

Backend: Flask, SQLAlchemy.

База даних: SQLite.

Документація API: Swagger.

Користувач → Frontend → Flask Routes → SQLAlchemy → SQLite DB
